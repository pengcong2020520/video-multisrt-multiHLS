# AI 短剧多语种翻译与音轨外切平台 BRD

## 1. 背景

短剧内容出海需要快速完成多语种本地化，包括字幕翻译、配音、音轨替换和预览验收。现阶段如果完全依赖人工翻译、人工配音和人工后期，交付周期长、成本高、批量产能不足。

本项目 MVP 面向内部运营人员，目标是用 AI 工具和成熟开源音视频组件搭建一条可落地的生产链路：上传原视频后，自动生成多语种外挂字幕和多语种替换人声音轨，并在网页中完成预览和下载。

## 2. 业务目标

| 目标 | 说明 |
| --- | --- |
| 提升处理效率 | 将单条 1-3 分钟短剧的初稿本地化产出从人工小时级压缩到分钟级。 |
| 降低试错成本 | 支持快速生成英文、西语、葡语等版本，用于内部审核和海外渠道测试。 |
| 标准化产物 | 输出可复用的字幕、音轨、预览视频和处理记录，方便后续进入剪辑或发布流程。 |
| 保留人工控制 | 支持人工校对台词、翻译和配音配置，避免纯自动生成导致不可控。 |
| 模型可替换 | ASR、翻译、TTS、声源分离均通过适配层接入，避免绑定单一供应商。 |
| 能力可复用 | 通过 Agent Runtime 编排独立 Skill，让媒体处理、翻译、配音、混音能力可独立复用和迭代。 |

## 3. 目标用户

### 3.1 核心用户

内部短剧运营人员。

### 3.2 用户特征

- 需要批量处理短剧视频。
- 关注交付速度、可预览性和结果可下载。
- 不一定具备音视频工程能力。
- 能接受 AI 初稿，但需要可人工校对和重跑。

## 4. 典型业务场景

### 场景 A：中文短剧出海

运营上传中文无字幕短剧，选择英文、西语、葡语目标语言。系统自动识别中文对白，生成本地化翻译、目标语字幕和目标语配音音轨。运营在网页中切换字幕和音轨预览，确认后下载结果包。

### 场景 B：英文原片多语种本地化

运营上传英文真人短剧，选择中文、西语、葡语目标语言。系统生成对应语言字幕和配音音轨，保留原视频 BGM，仅替换人声。

### 场景 C：局部修正和重跑

运营发现某一句翻译不符合短剧语感，直接修改翻译文本并重新生成该目标语言的字幕和局部配音，不需要整条视频从头处理。

## 5. MVP 范围

### 5.1 必须支持

- 上传 1-3 分钟 MP4/MOV 视频。
- 原视频可能是中文或英文，无内嵌字幕。
- Web 前端触发 Agent Runtime，并展示 Agent 编排进度。
- 自动提取音频。
- 自动分离人声和背景音，尽量保留 BGM/环境声。
- 自动 ASR，生成原语言台词、时间戳和可选说话人标记。
- 使用 AI 翻译生成短剧本地化风格翻译。
- 支持目标语言：英文、中文、西语；葡语作为可选增强。
- 生成外挂字幕：SRT 和 WebVTT。
- 生成目标语言人声音轨，并与原背景音混合成目标语言音轨。
- 网页内预览视频，支持切换字幕和音轨。
- 下载字幕、音轨、预览 MP4 或结果压缩包。
- 支持人工编辑原文、译文、角色音色、语速和重跑任务。

### 5.2 可延后

- 唇形级口型同步。
- 完整音色克隆工作流。
- 批量上传和队列优先级。
- 自动内容审核。
- 多人协作审批流。
- 账号权限体系的精细化管理。
- 视频发布到外部平台。
- 移动端管理后台。

### 5.3 明确不做

- 不做 C 端开放创作者平台。
- 不做完整剪辑软件。
- 不做人脸驱动、口型驱动、画面重生成。
- 不让 Agent 自主发布、删除、覆盖最终产物。
- 不保证重叠对白、强噪声、强混响、背景音乐压过人声时的完美处理。
- 不直接承诺商用版权合规，用户必须保证素材、声音和模型使用权合法。

## 6. 成功指标

| 指标 | MVP 目标 |
| --- | --- |
| 单条视频处理成功率 | 常规清晰对白视频 >= 80% |
| 单条 3 分钟视频初稿耗时 | <= 10 分钟，取决于模型和 GPU/API 资源 |
| 字幕时间戳可用率 | 运营无需大范围手动调轴 |
| 翻译可用性 | 运营以局部修改为主，不需要整篇重译 |
| 音轨可预览性 | 网页可稳定切换原音轨和目标语言音轨 |
| 结果可下载性 | 每个目标语言均可下载字幕和音轨 |

## 7. 业务风险

| 风险 | 影响 | 缓解 |
| --- | --- | --- |
| 人声分离质量不稳定 | BGM 残留人声或人声失真 | 保留原始音轨，提供分离质量提示，允许仅生成字幕不生成配音。 |
| ASR 错误导致翻译错误 | 台词含义偏差 | 提供人工校对和局部重跑。 |
| 翻译不符合短剧语感 | 海外用户理解差或转化差 | 建立短剧本地化提示词、术语表、角色设定和风格模板。 |
| TTS 时长不匹配 | 配音与画面节奏错位 | 支持目标时长、语速调节、静音补齐和局部重跑。 |
| 模型供应商不稳定 | 任务失败或成本失控 | 模型适配层、配置化路由、任务可重试。 |
| 版权和声音授权 | 商用风险 | 平台保留授权确认和操作日志，不默认进行未经授权的音色克隆。 |

## 8. 里程碑

| 阶段 | 目标 | 交付 |
| --- | --- | --- |
| M0 原型验证 | 跑通单视频单语言链路 | 上传、ASR、翻译、字幕、TTS、预览 |
| M1 MVP | 支持多目标语言和人工校对 | 多字幕、多音轨、下载包、任务状态 |
| M2 生产增强 | 提升稳定性和局部重跑效率 | 批量、队列、质量评分、成本统计 |
| M3 平台化 | 支持团队协作和发布前审核 | 权限、审批、版本管理、渠道配置 |
